# 🔋 Nanfu Design Skill — Industrial-Grade AI Skill for Dark Corporate Frontends

**[🇺🇳 English]** **[🇨🇳 中文 ↓]**

## 🚀 What is this?

This is an **AI coding assistant skill** — a structured design system that turns any AI agent (Codex CLI, Claude Code, Cursor, etc.) into a **professional dark-themed corporate page designer**, built on the design DNA of NANFU Global's industrial brand identity.

> **⭐ From a battery brand to digital icon — industrial precision meets premium UI.**

## ✨ Features

| 🌓 **Breath Section Alternation** | Every dark section MUST be followed by light — creates visual rhythm across 10+ sections |
| :--- | :--- |
| 🎨 **Red & Black Color System** | Saturated --primary: #e60008 accent red + pitch-black #000 backgrounds + dark/light card variants |
| 🔤 **96px Hero Typography** | Massive rem-based scale (0.96rem = 96px) — never shrink the minimums |
| 🎬 **GSAP Scroll Animation Engine** | Per-character text split, counter animation, parallax hero, staggered card entrances |
| 🪟 **Glassmorphism & Particle Canvas** | Glass cards on dark sections + interactive battery-particle canvas with mouse response |
| 🌐 **International Ready** | Language selector, interactive product popups, purchase country selector |
| ✅ **12 Anti-Pattern Checks** | Blocking QA checklist — catches adjacent same-bg sections, white-on-light text, double ScrollTriggers |

## 🎯 Who is this skill for?

- Developers using **AI coding assistants** who want dark premium corporate UIs fast
- Designers creating **brand sites, personal portfolios, product landing pages**
- Anyone wanting a **dark aesthetic with GSAP-powered scroll magic**

## ⚡ How to use this skill

`ash
git clone https://github.com/Mariachcareycoco/nanfu-design-skill.git
`

Then tell your AI assistant:

> **"Use nanfu-design-skill to build a corporate brand page"**

The AI automatically applies all design tokens, component templates, section alternation rules, and QA checks.

## 🔧 Skill tech stack

- **Framework**: Vanilla JS (no React/Vue required)
- **Animation**: GSAP 3.12+ + ScrollTrigger + Lenis smooth-scroll
- **CSS**: rem-based (5.20833vw base), flexbox + grid
- **Structure**: SKILL.md (330+ lines of spec) + gents/ (AI config) + README.md + LICENSE

## 📊 By the numbers

- 🔄 **10+ alternating sections** with mandatory light/dark rhythm
- 🚫 **12 anti-pattern checks** blocking common mistakes
- 🎬 **8 animation types** — split text, stagger, counter, parallax, particle
- 🎨 **20+ design tokens** — colors, typography, spacing, glass, shadows
